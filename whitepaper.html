Features

These are lists of some core features Lumeer has. The lists are not extensive so if you think we miss your favorite feature, do not hesitate to ask for it!

Click on the small question marks to learn more. Please stay tuned, we will be adding details to all of the items in the lists.

Views

Views allow you to see the same data through various lenses depending on your current task. There is a wide palette of options and we keep adding more.

  • Search results
    You can search for any of your data like you would do in a general search engine. The data are then displayed in several tabs very similar to your favorite search engine - All, Document, Links, Views - depending on your query. Moreover, you always get hints from Lumeer while typing your query.
  • Smart data table
  • Details
  • Links explorer
  • Contingency table
  • Calendar
  • Map
  • Gantt chart
  • Wiki
  • Agile board
  • Chart
  • Mind map and graph
  • Data experimentor

Collaboration

We understand how important collaboration with your colleagues and friends is. Lumeer facilitates that in its features.

  • Projects and users management
  • Data versioning and history tracking
  • Fine grained access control
  • Comments and discussions on record level
  • Sharing of data and views

Business Processes

Computers are very good at large data processing and automation. Lumeer leverages this by providing easy to understand and use business processes. You can start of by creating your own checklist of tasks, add some automated data manipulation steps later, and maybe share with all your colleagues in the end.

  • Process triggers
  • Process versioning
  • Data events
  • E-mail events
  • Business rules
  • Data manipulation tasks
  • Human tasks

Data

These features are more or less expected from all applications working with data. We just want to assure you that Lumeer supports them.

  • Documents, collections, links
  • Smart hints for better data management

  • Batch operations
  • Data imports
  • External collections
  • Easy copy'n'paste
  • Special types - tags, categories, geolocation
  • Data migration
  • Sandboxes
  • Collection archives
  • Search results snapshots
  • Query expressions
  • REST API
